# To-Do Application

This is a classic to-do application built with:
- **Frontend**: Angular
- **Backend**: .NET (C#)
- **Database**: PostgreSQL
- **Containerization**: Docker

## Project Structure
```
to-do/
├── backend/          # .NET project
│   ├── TodoApi/      # API project
│   └── docker-compose.yml  # Docker Compose for PostgreSQL and API
└── frontend/         # Angular project
```
